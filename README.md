Lucas Michallat, Théo Fillon et Thibaud Muller vous présente la SAE 6 Développement d'un outil décisionnel

docker-compose up --build -d

docker compose up -d

arrêter le docker:
docker compose down

http://localhost:8501/

http://localhost:8080
